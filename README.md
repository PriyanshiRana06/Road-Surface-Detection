# Road-Surface-Detection

🚗 Road Condition Detection System (GUI Based)

This project is a GUI-based Road Condition Detection System developed using Python. It analyzes road images and classifies the road condition while providing key metrics such as confidence score and road quality index. The system aims to enhance road safety by identifying road conditions efficiently.

The application features a user-friendly graphical interface that allows users to input road images and receive real-time results. It detects road type (e.g., smooth or rough), classifies overall condition (Good/Poor), and displays a confidence percentage along with a calculated Road Quality Index.

✨ Key Features
GUI-based interactive interface
Road type and condition detection
Confidence score display
Road Quality Index calculation
Real-time output visualization
